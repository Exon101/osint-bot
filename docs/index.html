Why OSINT Bot?

Everything You Need for
Intelligence Gathering

Purpose-built for ethical security research, CTF competitions, and authorized investigations.

Inline Button UI

Navigate all 30+ commands through a clean, intuitive inline keyboard interface. No complex commands to memorize — just tap and explore.

Cross-Platform Search

Search usernames across 30+ platforms including GitHub, Reddit, X, Instagram, TikTok, YouTube, Facebook, LinkedIn, and Steam.

Privacy First

Built-in rate limiting, input sanitization, and audit logging. No sensitive data stored. All queries logged for accountability.

Proxy Rotation

Built-in proxy manager with sequential, random, and failover rotation modes. Supports HTTP, HTTPS, SOCKS4, and SOCKS5 proxies.

Lightweight & Fast

Minimal dependencies, async architecture, and smart caching ensure rapid responses. Deploy anywhere: VPS, Docker, Heroku, or HuggingFace Spaces.

Easy to Extend

Modular handler architecture makes it trivial to add new modules. Clean separation between API clients, handlers, and utilities.

Trust & Safety

Built with
Security in Mind

Every layer of the bot is designed with security best practices.

Input Validation

Every command input is sanitized and validated before processing. No injection attacks possible.

Rate Limiting

Sliding-window rate limiting (10 req/60s per user) prevents abuse and API quota exhaustion.

Audit Logging

All queries are logged to SQLite with timestamps for accountability. Passwords and sensitive data are never logged.

Sandboxed Execution

Code runner uses blacklisted commands, timeout limits, and output size restrictions to prevent misuse.

Environment Secrets

All API keys and tokens stored as environment variables. No hardcoded credentials in source code.

Ethical Disclaimers

Educational disclaimers present throughout the bot. Designed for authorized security research only.

VPS / Bare Metal

cp .env.example .env && nano .env
python main.py

Docker

docker build -t osint-bot .
docker run -d --env-file .env osint-bot
